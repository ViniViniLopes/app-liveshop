# Stitch Prompt — Shoppable Media Screen

Create a high-fidelity vertical social commerce screen for a LiveShop app.

Context:
- The layout must work for recorded videos, live streams and replays.
- The visual reference is a neon shopping/product card experience similar to TikTok Shop/Reels commerce.
- The product data comes from Bling ERP and is hydrated by Supabase.
- The UI must support tenant branding: logo, primary color, accent color, button radius, favicon and store name.
- The screen must be mobile-first and thumb-friendly.

Required elements:
- Fullscreen vertical video/live area
- Live badge when status is live
- Product floating card at bottom
- Product image, name, price, stock indicator and CTA
- Social actions on right side
- Share and affiliate link button
- Save product button
- Push reminder button when live is scheduled
- Subtle neon glow/pulse for live/product emphasis
- Smooth product card switch animation
- Accessible labels and clear hierarchy

Style:
- premium, modern, energetic
- dark background option
- neon accents only for highlights
- clean card with readable text
- no hard-coded brand; all brand tokens must be replaceable

Export:
- Prefer React/Tailwind structure for web reference
- Provide component hierarchy and reusable design tokens
