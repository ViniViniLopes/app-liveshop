# Implementation Roadmap — LiveShop Platform

This document defines the phases and tasks for the platform build.

## Phase 1: Foundation & Security (Current)
- [ ] Core Database Schema (Auth, Tenants, Members).
- [ ] RLS Security Audit & Alignment.
- [ ] Bling OAuth & Product Cache Setup.
- [ ] Basic Storefront Domain Routing (Host header).

## Phase 2: Sales & Commerce
- [ ] Checkout Integration (Pagar.me / Stripe).
- [ ] Payment Webhooks & Idempotency.
- [ ] Order Sync with Bling.
- [ ] Affiliate Click & Commission Logic.

## Phase 3: Media & Live
- [ ] Video Asset Management.
- [ ] Live Session Logic (Scheduled, Live, Replay).
- [ ] Mobile RTMPS Integration (Native Modules).
- [ ] Realtime Product Switching in Storefront.

## Phase 4: SaaS Operations
- [ ] Onboarding Wizard.
- [ ] Plans, Subscriptions & Metering.
- [ ] Super Admin Console (Error Recovery, Logs).
- [ ] Feature Flags & Beta Rollout.

## Phase 5: Growth & White-label
- [ ] Cloudflare DNS Automation.
- [ ] Social Autopost Engine.
- [ ] Push Notification Engine (Opt-in/Opt-out).
- [ ] LGPD Tools (Export/Delete).

## Phase 6: Polish & Release
- [ ] E2E Acceptance Testing.
- [ ] Performance Optimization (Budgets).
- [ ] App Store & Google Play Submission.
