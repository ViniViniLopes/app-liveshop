# Onboarding Wizard

The onboarding wizard guides the seller through store activation.

## Required steps

1. Create store.
2. Brand store.
3. Setup domain.
4. Connect Bling.
5. Connect payment.
6. Connect social accounts.
7. Create first media/live.
8. Share storefront.

Each step must write to `tenant_onboarding_progress` and produce events for activation analytics.
