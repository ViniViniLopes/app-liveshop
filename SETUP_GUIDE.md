# Liveshop Zen - Configurações de Ambiente

# 1. Copie o conteúdo abaixo para um novo arquivo chamado `.env` na raiz da pasta `apps/web`
NEXT_PUBLIC_SUPABASE_URL=https://your-project-id.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key-here
NEXT_PUBLIC_PLATFORM_DOMAIN=localhost:3000

# 2. No arquivo `apps/mobile/src/lib/supabase.ts`, substitua os valores pelas suas chaves reais.

# 3. No terminal, execute:
# pnpm install
# pnpm dev
